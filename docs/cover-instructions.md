# Cover Image Instructions

The cover image needs to be created manually.

## Specifications
- **Size**: 1920x1080px (16:9)
- **Format**: PNG
- **Filename**: `docs/cover.png`

## How to Create

### Option A — Screenshot + Canva
1. Open [reposense-blond.vercel.app](https://reposense-blond.vercel.app)
2. Analyze `expressjs/express`
3. Take a full-page screenshot of the results page
4. Open in Canva or Figma
5. Add text overlay: **"RepoSense"** + **"Powered by IBM Bob"**
6. Export as PNG at 1920x1080
7. Save as `docs/cover.png`

### Option B — Composite from Existing Screenshots
1. Use the 4 screenshots already in `docs/`:
   - `hero_page.png`
   - `loading_state.png`
   - `results_page.png`
   - `bobshell_terminal.png`
2. Arrange in a 2x2 grid in Canva/Figma
3. Add "RepoSense — From stranger to contributor in minutes" overlay
4. Add "Powered by IBM Bob" badge
5. Export as 1920x1080 PNG

## Deadline
⚠️ **Must be created before May 17, 2026** (hackathon submission)

This is the only fix that cannot be fully automated.
